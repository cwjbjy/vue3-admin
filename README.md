# vue3-admin

## Project setup
```
前端编译：npm install
服务端编译：
cd server
npm install
```

### Compiles and hot-reloads for development
```
运行前端：npm run serve
运行服务端:npm run dev
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
